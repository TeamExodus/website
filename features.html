    Clear All Button

      The clear all button has three positions, (Search Bar[Current], Right, Left). When pressed all the current running apps including ones that don't show in the Overview menu will be realeased from memory.

    Memory Bar

      The memory bar will show the amount of RAM in use on the device out of the total RAM aviable, keep note that the Exodus Firmware and android as a whole does comsume about ~30% of RAM even when nothing is running.

Best of Exodus

Morph Mode:
       ►   Allows changing user experience, and configuration settings to match that of another distribution of android.

Exodus Updater
       ►   Get latest nightly updates as soon as they are built, with automated installation

Optimization
       ►   Optimized and tuned for greatest performance, battery life, and stability with Java 8, and backend improvements done to professional standards
       ►   Kernel Improvements on the most popular supported devices

Overview Options
       ►   Toggle the Google search bar
       ►   Memory bar showing the amount of used RAM
       ►   Clear all button that can be placed to the bottom right, left, or next to the search bar

Navigation Bar Configuration
       ►   Edit navigation bar targets with up tap, long press, and double tap commands
       ►   Add Extra layouts
       ►   Color navigation bar targets
       ►   Configure up to 3 navigation ring targets

Power Menu Options
       ►   Toggle eight power menu targets

System Animation
       ►   Tune Speed and custom animations for transitions based off of AOKP's Animations

CyanogenMod's Theme Engine
       ►   Full CMTE support with many themes actively supporting Exodus and custom themes being made every day

All of CyanogenMod Features and Options
      ►   Using Cyanogenmod's Open Source code allows us to give the users all the settings they are used to, with our new features and improvements on their features.
